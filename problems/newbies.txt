# Newbies

Enabling colors and "nice" paths on the command line:

    {{lang:bash}}
    wget http://uwwebpub.com/tools/bash_profile.sh  \
        && cat bash_profile.sh >> ~/.bash_profile   \
        && rm bash_profile.sh                       \
        && source ~/.bash_profile

Example script foo:

    {{lang:bash}}
    wget http://uwwebpub.com/tools/newbies.sh  \
        && chmod u+x ./newbies.sh              \
        && ./newbies.sh
